# DBeaver Community AppImage

This project provides an automated CI workflow to build and release DBeaver Community edition as an AppImage.

## Features
- **Automatic Updates:** Checks for new versions daily.
- **Differential Updates:** Includes `.zsync` for fast updates.
- **Continuous Delivery:** Automatically creates GitHub Releases.

## Usage
The workflow runs automatically every day, but you can trigger it manually from the Actions tab.
